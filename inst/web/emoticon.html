Emoticons Data Set

Usage

data(emoticon)

Format

A data frame with 81 rows and 2 variables

Description

A dataset containing common emoticons (adapted from Popular Emoticon List).

Details

  • meaning. The meaning of the emoticon
  • emoticon. The graphic representation of the emoticon
